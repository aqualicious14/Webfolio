Section 2

Projects

01

The first project is a working login and registration system that uses PHP and MYSQL to handle the creation and storing of users data delivering a robust login and registration system. It facilitates user creation, login credential verification, and user data storage. Utilizing XAMPP for integration, it ensures seamless PHP-MySQL interaction. Additionally, it incorporates a gaming feature where users earn XP points by guessing numbers, eventually advancing levels. All data, including XP and levels, is managed within a MySQL database, enhancing user engagement and system functionality.

02

The second project is a simple python Artificial Intelligence, made so that I can understand the fundamentals of machine learning and AI. Utilizing libraries like Pandas, NumPy, and scikit-learn, it constructs a simple linear regression model. The model predicts the time required for tasks based on their difficulty level, aiding in time management. By visualizing the data and evaluating model performance, this project offers practical insights into the application of machine learning techniques in real-world scenarios.

03

The third project is a wordle clone made in C++. By leveraging fundamental programming concepts and system functionalities, it creates an interactive word-guessing experience. The program randomly selects a word from a provided dictionary and challenges the player to guess it within five attempts. Through dynamic console color changes and feedback mechanisms, players receive visual cues to aid their guesses. This project not only showcases C++ proficiency but also highlights the fusion of logic, user interaction, and visual elements in game development.

04

The fourth project uses HTML, CSS, and JavaScript bringing the classic game of Tic-Tac-Toe to life in a web environment. With intuitive user interface design and dynamic functionality, it offers players an engaging gaming experience directly in their browsers. Leveraging modern web technologies, including CSS Grid for layout and JavaScript for game logic, this project demonstrates the fusion of front-end development skills to create interactive and visually appealing web applications.

05

The fifth project is a python script that replicates the Pokedex from Pokemon. It provides users with various functionalities to explore and interact with Pokemon data. Leveraging CSV parsing and menu-driven interaction, it enables users to perform tasks like displaying Pokemon details, searching for specific types, filtering by stats, and even catching Pokemon. With dynamic functionalities like catching wild Pokemon based on weighted probabilities and interactive menu options, this script offers an immersive experience akin to the iconic Pokedex from the Pokemon universe.
